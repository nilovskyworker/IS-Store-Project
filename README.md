# Проект информационной системы магазина
Это проект для лабораторной работы по Git.
## Описание ИС
Информационная система для автоматизации работы магазина.
Функции: каталог товаров, корзина, оформление заказов.

## Основные функции системы
- Регистрация и авторизация пользователей
- Поиск и фильтрация товаров
- Управление корзиной
- Оформление заказа с выбором оплаты и доставки
